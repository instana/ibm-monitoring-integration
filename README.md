# Instana

## Config pack for ITM agents

This repository contains the config pack for IBM ITM V6 and ITCAM agents so they may be configured to send monitoring data to an Instana Host Agent. The config pack is available in the [Releases](https://github.com/instana/ibm-monitoring-integration/releases) section of this repository.

Refer to the [Integrating with IBM ITM V6](https://www.ibm.com/docs/en/instana-observability/1.0.315?topic=technologies-integrating-itm-v6) documentation for more details on how to enable and configure the ITM sensor.
